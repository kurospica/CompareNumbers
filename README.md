## 工夫したこと

- まず数字をランダムに割り当てようとすると、数字が被ってしまう可能性があるので、
　- 最初に１００までの数字を格納したリストを作成し、その100の数字をシャッフル。
　- そして１～３番目の数字をプレイヤー１，４～６の数字をプレイヤー２に割り当てることで実現できました。
- 3ラウンドの勝負があり、1ラウンド終わるごとに選択した数字と選択肢を削除するようにしました。
